# Hotel-Management-Console-App

## Description
This is a Python-based hotel management system developed as a micro-project. It allows users to manage room booking, payment, check-in, and check-out operations using a menu-driven approach.

## Features
- View available rooms
- Book rooms
- Payment system
- Check-in and check-out
- File handling for record storage
- Exception handling

## Technologies Used
- Python
- File Handling
- Data Structures (List, Tuple, Dictionary)
